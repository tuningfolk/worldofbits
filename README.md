# worldofbits
A world where you can build

## Installation
Clone the repository
```
git clone https://github.com/tuningfolk/worldofbits.git
cd worldofbits/web
```

Install dependencies
```
npm install
# or
yarn install
```

## Run
```
npx vite
```

## Controls
Move around: WASD Keys
Turn: Arrow Keys
Add block: 'J' Key
