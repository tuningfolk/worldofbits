# worldofbits
A world where you can build

## Controls
Move around: WASD Keys  
Turn: Arrow Keys  
Add block: 'J' Key  


## Functionalities
- [x] move around
- [ ] blocks
    - [x] add blocks
    - [ ] remove blocks
    - [ ] logic flow blocks
- [ ] logic gates
    - [ ] AND
    - [ ] OR
    - [ ] NOT
    - [ ] XOR
    - [ ] XNOR
    - [ ] NAND
    - [ ] NOR
- [ ] Copy-Paste/Save chunks of a build as template 
- [ ] save world to backend
- [ ] multiplayer support

## Installation
Clone the repository
```
git clone https://github.com/tuningfolk/worldofbits.git
cd worldofbits/web
```

Install dependencies
```
npm install
# or
yarn install
```

## Run
```
npx vite
```
